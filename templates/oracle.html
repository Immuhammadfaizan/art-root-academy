Oracle SQL and PL/SQL Programming

Registration Fee: 5000 PKR

Course Duration: 12 Weeks

Course Description: This course provides a comprehensive introduction to Oracle SQL and PL/SQL programming, covering fundamental to advanced concepts essential for database management and development.

Course Schedule

  • Week 1: Introduction to Oracle SQL
  • Week 2: Basic SQL Queries
  • Week 3: Functions in SQL
  • Week 4: Aggregate Functions and Grouping
  • Week 5: Joining Tables
  • Week 6: Subqueries and Set Operators
  • Week 7: DML and DDL Operations
  • Week 8: Introduction to PL/SQL Programming
  • Week 9: Cursors in PL/SQL
  • Week 10: Procedures, Functions, and Packages
  • Week 11: Triggers and Exception Handling
  • Week 12: Advanced PL/SQL Concepts and Review

Hands-On Project

During Week 12, students will design and implement a small database system using Oracle SQL and PL/SQL, applying all learned concepts including table creation, data manipulation, writing queries, procedures, and triggers.

Optional Weekly Assessments

Students can engage in weekly assignments, quizzes, or coding exercises to reinforce key concepts from each week. A final exam or project presentation will take place in Week 12.
